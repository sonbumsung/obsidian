
## 도커 동작

```bash
1. os 
	docker run -dit [os] bash
	docker attach [컨테이너ID OR 컨테이너NAME]
2. process
	docker run -d -p [outer port:inner port] httpd
	docker exec -it [container_id] /bin/bash
```
